# Getting Started

Odin React was developed with the use of NodeJS and Vite in mind.

<blockquote>It's useful to already have your Odin Control instance running the
adapter(s) you wish you interact with using the GUI, before starting
development of the GUI. Whilst the GUI can run in development mode without
it, any components that try to interact with it will throw errors.</blockquote>

## Project Setup

A [Template](https://www.npmjs.com/package/@dssg/create-odin-react) is provided to
set up an Odin React project ready for development quickly and easily.

From within your Odin Control project, run the following command:
```bash
npm create @dssg/odin-react@latest
```

This will prompt you to provide a project Name, a root directory for the
GUI project, and some settings to ensure it connects to your Adapter automatically.

Simply follow the prompts to copy the templated project and get started.
The template script will by default automatically install the required
packages and start running the development server.

if you need to manually restart the development server, use the following command
from the GUI project's root:
```bash
npm run dev
```

Once running, the terminal will display the address of the served webpage. Vite
will also monitor the codebase and reload the server when changes are made.

```bash
# Terminal Output once running
  VITE v7.0.4  ready in 442 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```

Changes made to the codebase should show in that page, so it can
be used to test changes made quickly and easily.

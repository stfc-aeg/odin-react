# Welcome to Odin React

Odin React is a [React](https://react.dev/) package designed to make the development of GUIs for [Odin Control](https://odin-detector.github.io/odin-control/) interfaces easier and faster.

Using React, creating these GUIs is now simplified, with many of the provided Components automating the connection to the Odin Control API.

Many Odin React components are built upon the [Bootstrap React](https://react-bootstrap.netlify.app/) library, to provide styling and layout options.
This also means that Odin React Apps are automatically responsive to allow them to viewable on multiple sizes of screen.


## Storybook

This documentation has been generated using [Storybook](https://storybook.js.org/), providing interactive demonstrations of each component in the library.
Each page displays one or more examples of how to use the Component, as well as a table of *Props* that may be passed to the component.

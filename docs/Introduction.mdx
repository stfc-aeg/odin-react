            ### React
            React is a Javascript based Library that allows for the creation of
            Components: individual reusable UI elements that can contain their
            own state and logic. These components return a form of markup called
            **JSX**, which allows for a mix of HTML and Javascript within a single
            object.

            ### Vite
            Vite is an open source build tool that provides compilation settings
            and a development environment. During development, Vite can run
            the GUI as a standalone web service, with *Hot Module Replacement*
            (live monitoring of code changes, refreshing if anything chances).
            
            Vite's compilation step bundles source code, image assets, and any
            dependencies together into a minified static web resource.

            <a className="sb-action-link" href="https://vite.dev/" target="_blank"
            >Learn more ></a>

            ### Bootstrap
            Much of the styling and layout of Odin React was inherited from
            React Bootstrap, a Component Library that provides several useful
            components, and utilities like Transition animations, overlays,
            Layout components, and stylesheets with many useful CSS variables.

            Many Odin React components are built on top of the components and
            utilities provided by React Bootstrap, so it's use for developing
            the GUI is highly recommended to keep the visuals standardised and
            cohesive.
